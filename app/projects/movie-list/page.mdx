# Movie list app (TMDB API) - React + TypeScript Project

This is a **Movie Explorer application** built with **React, TypeScript, and TMDB API**, featuring **popular movies, filters, search, and detailed pages**.  
It demonstrates **data fetching with React Query**, **API integration**, **clean component architecture**, and **modern frontend practices**.

---

## **Technologies Used**

- **React + TypeScript** - Component-based architecture with type safety.
- **React Router v6** - Navigation between Home and Movie Details pages.
- **React Query (TanStack Query)** - Data fetching, caching, and mutation.
- **Axios** - API client for TMDB requests.
- **Tailwind CSS + shadcn/ui** - Modern UI components and styling.
- **Vite** - Fast development and build tooling.
- **TMDB API** - For real movie data (popular, search, details).
- **Vercel** - Deployment and hosting.

---
